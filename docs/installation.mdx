## System Requirements

### Minimum Requirements

<CardGroup cols={2}>
  <Card title="Operating System" icon="computer">
    - Windows 10/11
    - macOS 10.15+
    - Linux (Ubuntu 20.04+)
  </Card>
  <Card title="Software" icon="download">
    - Node.js 18.0.0+
    - Python 3.8+
    - 4GB RAM (8GB recommended)
  </Card>
</CardGroup>

### For GPU Training

<Info>
  GPU is optional but recommended for SMALL and BASE templates
</Info>

- NVIDIA GPU with CUDA support
- 12GB+ VRAM for SMALL template
- 40GB+ VRAM for BASE template
- CUDA 11.8 or 12.1

## Installation Methods

<Tabs>
  <Tab title="npx (Recommended)">
    ### Using npx
    
    No installation needed! Run directly:
    
    ```bash
    npx create-llm my-project
    ```
    
    <Check>Best for one-time use or trying out create-llm</Check>
  </Tab>

  <Tab title="Global Install">
    ### Global Installation
    
    Install once, use everywhere:
    
    ```bash
    npm install -g create-llm
    ```
    
    Then create projects with:
    
    ```bash
    create-llm my-project
    ```
    
    <Check>Best if you'll create multiple projects</Check>
  </Tab>

  <Tab title="Docker">
    ### Docker Installation
    
    Zero local dependencies needed:
    
    ```bash
    # Pull the image
    docker pull ghcr.io/theaniketgiri/create-llm:latest
    
    # Create a project
    docker run -it -v $(pwd):/workspace ghcr.io/theaniketgiri/create-llm my-project
    ```
    
    <Check>Best for consistent environments and CI/CD</Check>
    
    See [Docker Guide](/guides/docker) for more details.
  </Tab>
</Tabs>

## Python Environment Setup

<Warning>
  Always use a virtual environment to avoid dependency conflicts
</Warning>

<Tabs>
  <Tab title="venv">
    ```bash
    # Create virtual environment
    python -m venv venv
    
    # Activate (Windows)
    venv\Scripts\activate
    
    # Activate (macOS/Linux)
    source venv/bin/activate
    
    # Install dependencies
    pip install -r requirements.txt
    ```
  </Tab>

  <Tab title="conda">
    ```bash
    # Create conda environment
    conda create -n my-llm python=3.11
    
    # Activate
    conda activate my-llm
    
    # Install dependencies
    pip install -r requirements.txt
    ```
  </Tab>
</Tabs>

## GPU Setup (Optional)

### CUDA Installation

<Steps>
  <Step title="Check GPU">
    Verify you have an NVIDIA GPU:
    ```bash
    nvidia-smi
    ```
  </Step>

  <Step title="Install CUDA">
    Download and install CUDA Toolkit:
    - [CUDA 11.8](https://developer.nvidia.com/cuda-11-8-0-download-archive)
    - [CUDA 12.1](https://developer.nvidia.com/cuda-downloads)
  </Step>

  <Step title="Install PyTorch with CUDA">
    ```bash
    # For CUDA 11.8
    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
    
    # For CUDA 12.1
    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
    ```
  </Step>

  <Step title="Verify Installation">
    ```python
    import torch
    print(torch.cuda.is_available())  # Should print True
    print(torch.cuda.get_device_name(0))  # Your GPU name
    ```
  </Step>
</Steps>

## Verify Installation

After installation, verify everything works:

```bash
# Check create-llm version
npx create-llm --version

# Check Python
python --version

# Check PyTorch
python -c "import torch; print(f'PyTorch {torch.__version__}')"

# Check CUDA (if using GPU)
python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}')"
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Node.js not found">
    Install Node.js from [nodejs.org](https://nodejs.org/)
    
    Verify installation:
    ```bash
    node --version
    npm --version
    ```
  </Accordion>

  <Accordion title="Python not found">
    Install Python from [python.org](https://www.python.org/downloads/)
    
    Make sure to check "Add Python to PATH" during installation
  </Accordion>

  <Accordion title="CUDA not detected">
    1. Verify NVIDIA drivers are installed: `nvidia-smi`
    2. Check CUDA installation: `nvcc --version`
    3. Reinstall PyTorch with correct CUDA version
  </Accordion>

  <Accordion title="Permission denied errors">
    On Linux/macOS, you may need to use `sudo`:
    ```bash
    sudo npm install -g create-llm
    ```
    
    Or use npx to avoid global installation
  </Accordion>
</AccordionGroup>
