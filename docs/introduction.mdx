## What is create-llm?

create-llm is a CLI tool that scaffolds production-ready LLM training projects in seconds. Think of it as `create-next-app` but for training custom language models from scratch.

```bash
npx create-llm my-awesome-llm
cd my-awesome-llm
python training/train.py
```

## Quick Example

Here's how easy it is to train your first LLM:

<Steps>
  <Step title="Create Project">
    ```bash
    npx create-llm my-llm --template nano
    ```
  </Step>
  <Step title="Add Data">
    ```bash
    echo "Your training text" > data/raw/train.txt
    ```
  </Step>
  <Step title="Train">
    ```bash
    python training/train.py
    ```
  </Step>
  <Step title="Chat">
    ```bash
    python chat.py
    ```
  </Step>
</Steps>
